# 🎨 PaletteForge

A fast and interactive color generator built with modern web technologies.  
Generate beautiful color palettes instantly and improve your UI/UX workflow.

---

## 🚀 Features

- 🎲 Generate random colors instantly
- 📋 One-click copy HEX color codes
- ⚡ Lightweight and fast
- 📱 Fully responsive design
- 🎯 Clean and minimal UI

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- (Add React / Tailwind if you used them)

---

## 💡 How It Works

Click the generate button to instantly create a new color palette.  
Click any color to copy its HEX code to your clipboard.
